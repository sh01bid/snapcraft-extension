# Chrome Web Store Listing

## Extension Name
Screenshot King - Web Capture & Screen Recorder

## Short Description (132 chars max)
Capture screenshots, record your screen, and annotate with a powerful built-in editor. Export as PNG, JPEG, WebP, or MP4.

## Detailed Description

Screenshot King is your all-in-one screenshot and screen recording tool for Chrome. Capture, annotate, and share — all without leaving your browser.

🎯 SCREENSHOT MODES
• Capture Visible Area — Instantly grab what's on screen
• Capture Full Page — Scroll-capture entire web pages
• Select Region — Drag to select a custom area

🎬 SCREEN RECORDING
• Record Current Tab — Capture tab activity with audio
• Record Desktop — Record your entire screen or a window
• Export as WebM or MP4 (H.264) with real-time conversion

✏️ ANNOTATION EDITOR
• Draw freehand with customizable colors and thickness
• Add text annotations with adjustable font size
• Draw rectangles, circles, and arrows
• Blur sensitive areas for privacy
• Crop and resize with precision
• Undo/Redo with full history

📋 SMART FEATURES
• Auto-copy screenshots to clipboard
• Keyboard shortcuts for quick capture (Alt+Shift+V/F/S)
• Right-click context menu for instant access
• Desktop notifications on capture completion
• Customizable filename patterns with date/time variables

📁 CAPTURE HISTORY
• Browse all screenshots and recordings in a gallery view
• Search and filter by type (screenshots/recordings)
• Batch select, download, and delete
• Quick preview with metadata (size, date, dimensions)

⚙️ CUSTOMIZABLE SETTINGS
• Image format: PNG, JPEG, WebP
• Image quality slider (1-100)
• Video format: WebM (VP9) or MP4 (H.264)
• Recording quality: 720p / 1080p / Original
• Frame rate: 15/24/30/60 FPS
• Recording countdown timer
• System audio & microphone toggle
• Dark theme
• Max history items control

🌍 MULTILINGUAL
• English and Chinese (Simplified) interface

🔒 PRIVACY
• All data stays local — no cloud uploads, no tracking
• No account required
• Open source

# Chrome Web Store 商店信息（中文）

## 扩展名称
截图王 - 网页截图与录屏工具

## 简短描述（132字符以内）
截图、录屏、标注一站式工具。支持全页截图、区域选取、屏幕录制，内置标注编辑器，导出 PNG/JPEG/WebP/MP4。

## 详细描述

截图王 是一款功能强大的 Chrome 截图与录屏扩展。截图、标注、分享，全部在浏览器内完成。

🎯 截图模式
• 截取可见区域 — 一键截取当前屏幕内容
• 截取整页 — 自动滚动截取完整网页
• 选取区域 — 拖拽选择自定义截图范围

🎬 屏幕录制
• 录制当前标签页 — 捕获标签页活动及音频
• 录制桌面 — 录制整个屏幕或指定窗口
• 导出 WebM 或 MP4 (H.264) 格式，支持实时转换

✏️ 标注编辑器
• 自由画笔 — 可自定义颜色和粗细
• 文字标注 — 可调整字体大小
• 矩形、圆形、箭头绘制
• 马赛克模糊 — 保护隐私信息
• 精准裁剪和缩放
• 完整的撤销/重做历史

📋 智能功能
• 截图自动复制到剪贴板
• 快捷键一键截图 (Alt+Shift+V/F/S)
• 右键菜单快速访问
• 截图完成桌面通知提醒
• 自定义文件名模式（支持日期/时间变量）

📁 历史记录
• 画廊式浏览所有截图和录制
• 按类型搜索和筛选（截图/录制）
• 批量选择、下载、删除
• 快速预览及元信息查看

⚙️ 丰富的设置选项
• 图片格式：PNG、JPEG、WebP
• 图片质量滑块 (1-100)
• 视频格式：WebM (VP9) 或 MP4 (H.264)
• 录制画质：720p / 1080p / 原始分辨率
• 帧率：15/24/30/60 FPS
• 录制倒计时
• 系统声音和麦克风开关
• 深色主题
• 历史记录数量上限设置

🌍 多语言支持
• 中文（简体）和英文界面

🔒 隐私保护
• 所有数据本地存储 — 无云端上传、无追踪
• 无需注册账号
• 开源免费
